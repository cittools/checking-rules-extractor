Multiple variable declarations shall not be allowed on the same line [JSF-152-2]

DESCRIPTION 

Rule triggers if some variables are declared on the same line.
See also: FORMAT-33
BENEFITS 

Rule increases readability and prevents confusion.
EXAMPLE 

void foo( ) {
    int a, b;   // Violation
}
REPAIR 

void foo( ) {
    int a;      // OK
    int b;      // OK
}
