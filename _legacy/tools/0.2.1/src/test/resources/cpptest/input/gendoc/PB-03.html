Using mixed types in ternary operator is not allowed [PB-03-3]

DESCRIPTION 

Do not use mixed types in ternary operators.
BENEFITS 

Rule prevents implicit conversions which may result in a loss of information
and improves readability and maintainability of code.
EXAMPLE 

void foo( ) {
    int x;
    int y;
    x = ((y > 5) ? 1.1 : 0.2);      // Violation
}
REPAIR 

void foo( ) {
    int x;
    int y;
    x = ((y > 5) ? 1 : 0);      // OK
}
