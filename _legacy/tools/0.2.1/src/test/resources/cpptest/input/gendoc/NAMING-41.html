Header files will always have a file name extension of ".h" [NAMING-41-3]

DESCRIPTION 

Header files will always have a file name extension of ".h".
Rule checks if header files included by source file have a
file name extension of ".h".
See also: PREPROC-08
SINCE 

v7.1
BENEFITS 

Rule improves readability and maintainability of code.
EXAMPLE 

//file.cpp

#include "foo.hh"   // Violation
REPAIR 

//file.cpp

#include "foo.h"   // OK
REFERENCES 

JOINT STRIKE FIGHTER, AIR VEHICLE, C++ CODING STANDARDS
Chapter 4.9 Style, AV Rule 53
