All letters contained in function and variable names will be composed entirely of lowercase letters [NAMING-44-3]

DESCRIPTION 

All letters contained in function and variable names will be composed
entirely of lowercase letters.


Exceptions:

Constructors and destructors.
SINCE 

v7.1
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

class Sample_class
{
public:
    int Some_function();      // Violation
private:
    int Some_variable;        // Violation
};
REPAIR 

class Sample_class
{
public:
    int some_function();      // OK
private:
    int some_variable;        // OK
};
REFERENCES 

JOINT STRIKE FIGHTER, AIR VEHICLE, C++ CODING STANDARDS
Chapter 4.9 Style, AV Rule 51
