The goto statement shall not be used [MISRA2004-14_4-3]

DESCRIPTION 

The goto statement shall not be used.
BENEFITS 

The goto statement can lead to errors and confusion.
EXAMPLE 

int foo( int a ) {
    if (a < 0) {     
        goto end;       /* Violation */
    } 
    a = foo(a-1);
end:
    return a+1;
}
REPAIR 

int foo( int a ) {
    /* Code was changed and does not use goto anymore */
    if (a >= 0) {
        a = foo(a-1);
    }
    return a+1;
}
