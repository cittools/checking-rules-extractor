Declare objects at function scope [JSF-136-4]

DESCRIPTION 

Declarations of objects should be at function scope
unless a wider scope is necessary.
BENEFITS 

Prevents using global variables.
EXAMPLE 

int globalVar;  // Violation
REPAIR 

void foo( ) {
    int localVar;   // OK 
}
