Make declarations at file scope static where possible [JSF-137-4]

DESCRIPTION 

All declarations at file scope should be static where possible.
BENEFITS 

Prevents from accidentally overwriting of functions or variables
EXAMPLE 

int g1; /* Violation */
void foo1() {}; /* Violation */
REPAIR 

static int g2; /* OK */
static void foo2(){}; /* OK */
