The static storage class specifier shall be used in definitions and declarations of objects and functions that have internal linkage [MISRA2004-8_11-3]

DESCRIPTION 

"It is good practice to apply the static keyword consistently to 
all declarations of objects and functions with internal linkage."
BENEFITS 

Rule improves good programming style and readability.
EXAMPLE 

/* file.h */
static int x;

/* file.c */
#include "file.h"
extern int x;        /* Violation */
REPAIR 

/* file.h */
static int x;

/* file.c */
#include "file.h"
static int x;         /* OK */

1. MISRA-C:2004 Guidelines for the use of the C language in critical systems
