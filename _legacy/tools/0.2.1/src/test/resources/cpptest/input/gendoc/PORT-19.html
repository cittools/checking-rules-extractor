Instead of casting a long type operand of an expression to int cast the result of the expression [PORT-19-3]

DESCRIPTION 

This rule checks whether you use explicit cast on operands of type long. 
Explicit narrowing casts of type long should be used on expressions,
not operands.
BENEFITS 

By following this rule, you can prevent data loss.
EXAMPLE 

void foo() 
{
   int int1, int2;
   long longValue;
   int2 = ((int)longValue)/int1; // Violation
}
REPAIR 

void foo() 
{
   int int1, int2;
   long longValue;
   int2 = (int)(longValue/int1); // OK
}
