Do not assign function return value to a variable of incompatible type [PB-07-3]

DESCRIPTION 

Do not assign function return value to a variable of incompatible type.
BENEFITS 

Rule prevents implicit conversions which may result in a loss of information
and improves readability and maintainability of code.
EXAMPLE 

float goo();
void foo()
{
  int a = 0;
  a = goo(); // Violation
}
REPAIR 

float goo();
void foo()
{
  float a = 0.1;
  a = goo(); // OK
}
