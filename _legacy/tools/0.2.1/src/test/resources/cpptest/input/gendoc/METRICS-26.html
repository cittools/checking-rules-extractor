Source lines will be kept to a length of 120 characters or less [METRICS-26-3]

DESCRIPTION 

This rule checks that physical lines are 120 or less characters.
SINCE 

v7.1
NOTES 

Tab is counted as one character.
BENEFITS 

Readability and style.
Very long source lines can be difficult to read and understand.
EXAMPLE 

int       /* .......................................................................................*/  foo; // Violation
REPAIR 

int /* ... */  foo; // OK
REFERENCES 

JOINT STRIKE FIGHTER, AIR VEHICLE, C++ CODING STANDARDS
Chapter 4.9 Style, AV Rule 41
