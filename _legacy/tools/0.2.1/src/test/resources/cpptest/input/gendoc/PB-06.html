Do not assign incompatible variable types [PB-06-3]

DESCRIPTION 

Do not assign incompatible variable types.
See also: MISRA-043, PORT-14
BENEFITS 

Rule prevents data loss and improves readability
and maintainability of code.
EXAMPLE 

void foo()
{
    int a = 0;
    float b = 1.1;
    a = b; // Violation 
    b = a; // Violation
}
REPAIR 

void foo()
{
    int a = 0;
    int b = 1;
    a = b; // OK
    b = a; // OK
}
