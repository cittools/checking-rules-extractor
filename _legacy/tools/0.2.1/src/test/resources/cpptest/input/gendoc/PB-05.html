Return value of a function must match declared return type [PB-05-3]

DESCRIPTION 

Return value of a function must match declared return type.
BENEFITS 

Rule prevents implicit conversions which may result in a loss of information
and improves readability and maintainability of code.
EXAMPLE 

typedef int int_t;
typedef float float_t;

int_t foo( ) {
    float_t ret = 0;
    return ret;            // Violation
}
REPAIR 

typedef int int_t;
typedef float float_t;

int_t foo( ) {
    int_t ret = 0;
    return ret;            // OK
}
