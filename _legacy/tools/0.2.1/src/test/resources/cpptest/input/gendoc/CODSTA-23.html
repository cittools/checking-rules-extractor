All 'if' statements should have an 'else' clause [CODSTA-23-3]

DESCRIPTION 

This rule checks if each of your 'if' statements has an 'else' clause.
See also: MISRA2004-14_10
BENEFITS 

Writing 'if' and 'else' in concert improves readability and reliability.
EXAMPLE 

void Foo(bool b) {
  int i = 0;
  int j = 0;
  if (b) {       // Violation
    i += j;
  }
}
REPAIR 

void Foo(bool b) {
  int i = 0;
  int j = 0;
  if (b) {            
    j += i;
  } else {	// OK
    i++;
  }
}
