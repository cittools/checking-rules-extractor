Redundant cast to the same type is not allowed [OPT-29-4]

DESCRIPTION 

Avoid redundant casts to the same type.
Rule reports a violation message when an expression's
type is casted to the same type.
SINCE 

v7.2
BENEFITS 

Rule prevents using redundant casts.
EXAMPLE 

int someFunction();

void foo(int p)
{
    /* ... */
    p = (int)someFunction();  // Violation
}
REPAIR 

int someFunction();

void foo(int p)
{
    /* ... */
    p = someFunction();  // OK
}
