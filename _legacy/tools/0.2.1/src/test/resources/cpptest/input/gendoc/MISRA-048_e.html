Use explicit type conversions for arithmetic using signed and unsigned values [MISRA-048_e-3]

DESCRIPTION 

"Use explicit type conversions for arithmetic using signed and unsigned values."
BENEFITS 

Complying with this rule leads to safer and more readable code.
EXAMPLE 

void foo(signed char a)
{
    signed char b = 'b';
    unsigned char c = 'c';
    a = b + c;                 // Violation
}
REPAIR 

void foo(signed char a)
{
    signed char b = 'b';
    unsigned char c = 'c';
    a = (unsigned char)b + c;  // OK
}
