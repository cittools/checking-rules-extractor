An include file should not contain more than one class definition [PFO-03-3]

DESCRIPTION 

"An include file should not contain more than one class definition."
NOTES 

Rule does not count nested classes.
BENEFITS 

Rule makes source code more readable.
EXAMPLE 

// file.hh - header file
// Violation
class testClass1{};
class testClass2{};
REPAIR 

// file.hh - header file
// OK - only one class definition
class testClass1{};
