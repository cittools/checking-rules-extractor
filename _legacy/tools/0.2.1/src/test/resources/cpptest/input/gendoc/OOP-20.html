Avoid unnecessary explicit cast from derived to a base class [OOP-20-3]

DESCRIPTION 

Avoid unnecessary explicit cast from derived to a base class.
This rule checks if your code contains unnecessary casts.
BENEFITS 

It is unnecessary to explicitly upcast to a base class.
Avoid unnecessary casts for legibility.
EXAMPLE 

class Base { /* ... */ };
class Derived : public Base { /* ... */ };
void Func() {
   Derived *d = new Derived();
   Base *b = (Base *)d;		// Violation
}
REPAIR 

class Base { /* ... */ };
class Derived : public Base { /* ... */ };
void Func() {
   Derived *d = new Derived();
   Base *b = d;			// OK
}
