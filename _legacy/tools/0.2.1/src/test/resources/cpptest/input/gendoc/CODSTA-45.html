Parameters and variables of class/parent classes/parent structs should have different name [CODSTA-45-1]

DESCRIPTION 

Rule checks if parameters and variables of class/parent classes/
/parent structs have the same name (case sensitive). 
If so, then it reports violation message.
See also: CODSTA-44
BENEFITS 

Rule improves readability and prevents writing ambiguous code.
EXAMPLE 

class Base {
public:
    int member;
    float var1;
    float var2;
    short var3;

    void foo( double var3 ) {  // Violation
        float member;    
    }
};
REPAIR 

class Base {
public:
    int member;
    float var1;
    float var2;
    short var3;

    void foo( double var4 ) {   // OK
        float member;    
    }
};
