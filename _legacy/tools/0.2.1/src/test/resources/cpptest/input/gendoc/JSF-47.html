Do not use identifiers which begin with one or two underscores (`_' or `__') [JSF-47-3]

DESCRIPTION 

"Do not use identifiers which begin with one or two underscores (`_' or `__')."
BENEFITS 

Rule prevents undefined behaviour.
EXAMPLE 

void _foo( );  // Violation
REPAIR 

void foo( );   // OK
