The library functions atof, atoi and atol from library <cstdlib> shall not be used [MISRA2008-18_0_2-3]

DESCRIPTION 

'atof', 'atoi' and 'atol' functions from library <stdlib.h> have undefined
behaviour associated with them when the string cannot be converted.
BENEFITS 

Prevents using functions which have sometimes undefined behaviour.
EXAMPLE 

#include <stdlib.h>
void foo( void ) {
    char *s; double x; int i; long l;
    s = "  -2309.12E-15";    
    x = atof( s );           /* Violation */
    s = "  -9885 pigs";      
    i = atoi( s );           /* Violation */
    s = "98854 dollars";     
    l = atol( s );           /* Violation */
}
REPAIR 

Do not use atof, atoi and atol functions.
