Struct type name shall begin with an uppercase letter prefixed by software element and suffixed by '_t' [NAMING-12-3]

DESCRIPTION 

Struct type name shall begin with an uppercase letter prefixed by software
element and suffixed by '_t'.
See also: NAMING-09, NAMING-10, NAMING-11
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

struct LOC_Position {};	// Violation
REPAIR 

struct LOC_Position_t {};	// OK
REFERENCES 

Recommended by ParaSoft
