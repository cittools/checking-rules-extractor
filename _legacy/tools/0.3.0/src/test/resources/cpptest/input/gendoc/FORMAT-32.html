The dereference operator `*' and the address-of operator `&' should be directly connected with the type names [FORMAT-32-3]

DESCRIPTION 

"The dereference operator * and the "address-of" operator & should be
directly connected to the type names in declaration and definition."
BENEFITS 

Rule makes source code more readable.
EXAMPLE 

void foo() {
    int k=42;
    int *p1;	// Violation
    int &r1 = k;// Violation
}
REPAIR 

void foo() {
    int k=42;
    int* p2;	// OK
    int& r2 = k;// OK
}
