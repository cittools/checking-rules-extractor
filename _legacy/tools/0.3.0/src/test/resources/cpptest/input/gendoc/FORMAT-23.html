There shall be no white spaces between the "return" or "sizeof " statements and its opening parenthesis [FORMAT-23-3]

DESCRIPTION 

This rule checks for white spaces between the "return" and "sizeof" statements.
See also: FORMAT-24, FORMAT-25
BENEFITS 

Improves the readability of code.
EXAMPLE 

void foo() 
{
   sizeof (int); // Violation
}
REPAIR 

void foo() 
{
   sizeof(int); // OK
}
