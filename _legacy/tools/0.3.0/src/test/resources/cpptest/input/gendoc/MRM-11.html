Always assign a new value to a pointer that points to deallocated memory [MRM-11-3]

DESCRIPTION 

"Always assign a new value to a pointer that points to deallocated memory."
All pointers should be set to either 0 or to a valid address immediately
after delete statement.
See also: MRM-06, MRM-08, MRM-09, MRM-10, MRM-30
BENEFITS 

Rule prevents accessing a pointer or a reference to a deleted object.
EXAMPLE 

void foo( ) {
    int* p;
    int* q;
    delete p;	// Violation - set deleted pointer to 0 in next expression
    delete q;	// Violation - set deleted pointer to 0 in next expression
    p = 0;
    q = new int;
}
REPAIR 

void foo( ) {
    int* p;
    int* q;
    delete p;	// OK
    p = 0;
    delete q;	// OK
    q = new int;
}
