Do not cast an int pointer to a long pointer [JSF-181_c-3]

DESCRIPTION 

"Do not assume that int and long have the same size."
Do not cast an int pointer to a long pointer. 
These are not compatible in 64 bit, even though in 32 bit
they would have been interchangeable.
See also: PORT-05, PORT-06, PORT-07, PORT-08, PORT-10, PORT-12
BENEFITS 

Rule prevents loss of data.
EXAMPLE 

void f2005() {
    int *intPointer;
    long *longPointer;
    longPointer = (long *)intPointer;	// Violation
}
REPAIR 

Do not cast an int pointer to a long pointer.
