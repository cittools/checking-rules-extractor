Whenever an object is declared or defined, its type shall be explicitly stated [MISRA2004-8_2_b-3]

DESCRIPTION 

Whenever an object is declared or defined, 
its type shall be explicitly stated.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

extern x;  /* Violation - Non-compliant - implicit int type */
const y;   /* Violation - Non-compliant - implicit int type */
REPAIR 

typedef int int16_t;
extern int16_t x;       /* OK - explicit type */
const int16_t y;        /* OK - explicit type */
REFERENCES 

MISRA-C:2004 Guidelines for the use of the C language in critical systems
Chapter 6, Section 8
