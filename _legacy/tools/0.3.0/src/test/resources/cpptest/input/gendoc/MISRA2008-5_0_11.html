The plain char type shall only be used for the storage and use of character values [MISRA2008-5_0_11-3]

DESCRIPTION 

"The plain char type shall be used only for the storage and use of 
character values.
The only permissible operators on plain char types are assignment 
and equality operators (=, ==, != )"
BENEFITS 

Rule makes the code more readable and less confusing.
EXAMPLE 

char a = 65;     /* Violation */
void foo( ) {
    a = 66;     /* Violation */
    a++;        /* Violation */
    if (a < 67) /* Violation */ 
    {
    }
}
REPAIR 

char a = 'A';     /* OK */
void foo( ) {
    a = 'B';      /* OK */
    if (a == 'C') /* OK */ 
    {
    }
}
REFERENCES 

1. MISRA-C:2004 Guidelines for the use of the C language in critical systems
   Chapter 6, Section 6

2. MISRA C++:2008 Guidelines for the use of the C++ language in critical
   systems, Chapter 6, Section 5, Rule 5-0-11
