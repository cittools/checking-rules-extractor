Do not initialize unsigned integer variables with signed constants [INIT-02-3]

DESCRIPTION 

Do not initialize unsigned integer variables with signed constants.
Numeric constants have a type.
Use the appropriate suffix to avoid undesirable implicit cast.
Rule triggers only when a value of constant is smaller than zero. 
See also: PB-08,  MISRA2004_10_1_h
BENEFITS 

Rule prevents undesirable implicit casts.
EXAMPLE 

void foo()
{
   unsigned int y = -21;	// Violation
}
REPAIR 

void foo()
{
   signed int y = -21;	// OK
}
