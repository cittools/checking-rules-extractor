Opening curly brace '{' shall appear on a line by themselves [FORMAT-02-3]

DESCRIPTION 

This rule checks that open curly braces "{" appear on a line by themselves. 
See also: FORMAT-03, FORMAT-34


Exceptions:

Rule does not adhere to enums, initializations and empty blocks.
enum E {EN1 = 2, EN2 = 1};  // OK
int array[] = {1,2,3,4,5};  // OK
class FooBar{};             // OK
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

class A1
{ int a;                      // Violation
};

struct S1 {                   // Violation
    int a;
};
REPAIR 

class A1
{
    int a;                    // OK
};

struct S1
{                             // OK
    int a;
};
