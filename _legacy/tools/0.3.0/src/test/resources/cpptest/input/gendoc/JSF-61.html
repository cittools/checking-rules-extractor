Closing curly brace '}' shall appear on a line by themselves [JSF-61-3]

DESCRIPTION 

This rule checks that curly braces "}" appear on a line by themselves. 
See also: FORMAT-02, FORMAT-34


Exceptions:

Rule does not adhere to enums, initializations and empty blocks.
enum E {EN1 = 2, EN2 = 1};  // OK
int array[] = {1,2,3,4,5};  // OK
class FooBar{};             // OK
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void goo()
{
    int a, c;
    if (c == 0)
    {
        a = 2;}   // Violation
}
REPAIR 

void goo()
{
    int a, c;
    if (c == 0)
    {
        a = 2;
    }            // OK
}
