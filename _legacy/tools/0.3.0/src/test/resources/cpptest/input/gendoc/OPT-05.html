Eliminate unused private member variables [OPT-05-3]

DESCRIPTION 

This rule checks for unused private member variables. 
This rule applies only to the C++ programming language.
BENEFITS 

Eliminating unused private member variables
improves legibility and increases efficiency.
EXAMPLE 

class Foo {
public:
   Foo(): x(0), y(0) { }
   int getX() {
       return x;
   }
private:
   int x;
   int y;  // Violation        
   int z;  // Violation
};
REPAIR 

class Foo {
public:
   Foo(): x(0), y(0), z(1) { }
   int getX() {
       return x;
   }
   int getY() {
       return y;
   }
   int getZ() {
       return z;
   }
private:
   int x;
   int y;  // OK
   int z;  // OK
};
