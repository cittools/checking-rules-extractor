Avoid functions with over 50 lines [METRICS-01-5]

DESCRIPTION 

Programs should be designed so that most of the functions will be smaller 
than 50 lines of source listening. Algorithms are easier to create and to
understand if they are built of pieces small enough to be grasped as one
concept.
See also: METRICS-20, METRICS-21, METRICS-22
BENEFITS 

Rule improves readability and maintainability of code.
