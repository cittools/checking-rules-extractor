Use capital 'L' instead of lowercase 'l' to indicate long [PORT-01-5]

DESCRIPTION 

Use capital 'L' instead of lowercase 'l' to indicate long.
BENEFITS 

Prevents data losing. Improves the readability of code.
EXAMPLE 

void func()
{
    int i = 0;
    i = 34l;  // Violation
}
REPAIR 

void func()
{
    int i = 0;
    i = 34L;  // OK
}
