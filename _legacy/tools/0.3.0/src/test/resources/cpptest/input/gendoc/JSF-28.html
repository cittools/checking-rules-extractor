The #ifndef and #endif pre-processor directives will only be used to prevent multiple inclusions of the same header file [JSF-28-3]

DESCRIPTION 

The #ifndef and #endif pre-processor directives will only be used
as defined in below example to prevent multiple inclusions of the
same header file.

#ifndef Header_filename
#define Header_filename
// Header declarations...
#endif
SINCE 

v7.1
BENEFITS 

Conditional code compilation should be kept to a minimum as it can
significantly obscure testing and maintenance efforts.
EXAMPLE 

#ifndef MAX                 // Violation

int max = 10;

#endif                      // Violation

int a;
REPAIR 

#ifndef FOO_H               // OK
#define FOO_H 

int max = 10;
int a;

#endif                      // OK
