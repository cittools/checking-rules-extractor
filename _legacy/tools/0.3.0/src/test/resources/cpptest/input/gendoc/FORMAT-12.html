There shall be a single ASCII space character between a conditional statement keyword and its opening parenthesis [FORMAT-12-3]

DESCRIPTION 

This rule checks whether there is a single ASCII space character 
between a conditional statement keyword and its opening parenthesis.
See also: FORMAT-13
BENEFITS 

This promotes readability by providing a clear indication that a statement
is a conditional statement versus a function, and by providing a clear
break between the conditional keyword and its associated expression.
EXAMPLE 

void foo()
{
   int x=1;
   int y=1;
   
   if(x == y) // Violation
   {
   }
}
REPAIR 

void foo()
{
   int x=1;
   int y=1;

   if (x == y) // OK
   {
   }
}
