The increment expression in a for loop will perform no action other than to change a single loop parameter to the next value for the loop [CODSTA-53-3]

DESCRIPTION 

The increment expression in a for loop will perform no action other
than to change a single loop parameter to the next value for the loop
SINCE 

v7.1
BENEFITS 

Rule improves readability of code.
EXAMPLE 

void foo()
{
    int k = 0;

    for(;k>0; ){}                 // Violation
    for(int i=0; k>0; k--, i++){} // Violation
    for(k--; k>0; k+1){}          // Violation
}
REPAIR 

void zoo(){}

class A
{
public:
    int i;
};

void foo()
{
    int k = 0;
    A obj;
    
    for(int i = k+2-10; k>0; zoo()){} // OK
    for(k=2; k>0; k--){}              // OK
    for(int i =0; i>0; i=i-1){}       // OK
    for(k = obj.i; k>0; k-=1){}       // OK
}
