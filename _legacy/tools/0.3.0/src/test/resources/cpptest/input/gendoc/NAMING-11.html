Enumerated type name shall begin with an uppercase letter prefixed by software element and suffixed by '_t' [NAMING-11-3]

DESCRIPTION 

Enumerated type name shall begin with an uppercase letter prefixed by software
element and suffixed by '_t'.
See also: NAMING-09, NAMING-10, NAMING-12
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

enum LOC_PossibleColors { RED2, BLUE2, GREEN2 };     // Violation
REPAIR 

enum LOC_PossibleColors_t { RED, BLUE, GREEN };       // OK
REFERENCES 

Recommended by ParaSoft
