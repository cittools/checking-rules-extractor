A switch-label shall only be used when the most closely-enclosing compound statement is the body of a switch statement [MISRA2008-6_4_4-3]

DESCRIPTION 

"The scope of a case or default label shall be the compound statement, 
which is the body of a switch statement. 
All case clauses and the default clause shall be at the same scope."
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void foo( int a, int b ) {
    switch(a) {
    case 1:
        {
            case 6:     /* Violation */
            ;
            default:    /* Violation */
            break;
        }
        break;
    case 2:
        if (b == 1) {
             case 3:      /* Violation */
             break;
        }
        break;
    }
}
REPAIR 

Do not use nested 'case'/'default' statements.
