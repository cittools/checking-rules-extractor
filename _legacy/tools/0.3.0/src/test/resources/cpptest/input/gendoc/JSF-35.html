Use mechanism that prevents multiple inclusion of the file i.e. include guards or "#pragma once" preprocessor directive [JSF-35-3]

DESCRIPTION 

"Every include file must contain a mechanism that prevents multiple inclusion 
of the file."
The multiple inclusion mechanism should be either:

- #pragma once - preprocessor directive

or

- include guards defined as follows:
#ifndef FILENAME_H
#define FILENAME_H
// code
#endif

or

#if !defined(FILENAME_H)
#define FILENAME_H
// code
#endif

See also: PFO-02, PFO-07, and MISRA2004-19_15
SINCE 

v7.1
BENEFITS 

Rule improves maintainability and optimization of code.
EXAMPLE 

// file foo.hh
// Violation - no multiple inclusion mechanism present
REPAIR 

// file foo.hh
// OK
#pragma once
