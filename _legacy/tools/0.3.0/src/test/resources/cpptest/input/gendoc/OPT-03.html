Eliminate unused parameters [OPT-03-2]

DESCRIPTION 

This rule checks for unused parameters.
NOTES 

The rule does not report unused parameters for virtual 
functions overriding virtual functions in base class.
BENEFITS 

Eliminating unused parameters improves legibility and increases efficiency.
EXAMPLE 

int Foo(int i, int k) {      // Violation
    i = 5;
    return i;
}
REPAIR 

int Bar(int i, int j) {      // OK
    i += j;
    return i;
}
REFERENCES 

Scott Meyers, "More Effective C++: 35 New Ways to Improve
Your Programs and Designs", Addison Wesley, (31 Mar 1996), Item 6
