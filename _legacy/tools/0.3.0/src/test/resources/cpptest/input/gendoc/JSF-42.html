Only one statement shall be allowed per line [JSF-42-3]

DESCRIPTION 

This rule checks that there is only one statement per line.
BENEFITS 

This rule promotes readability and maintainability by reducing code complexity.
EXAMPLE 

void foo()
{
   int i; char c; // Violation
}
REPAIR 

void foo()
{
    int ii; // OK
    char cc; 
}
