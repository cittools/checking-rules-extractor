Do not cast a signed char to an unsigned int [PB-12-3]

DESCRIPTION 

This rule checks whether you cast a signed char to an unsigned int.
BENEFITS 

Prevents losing of precision by casting a signed char to an unsigned int.
EXAMPLE 

bool foo( char ch )
{
    if( (unsigned) ch == 0xFF ) {  // Violation
        return true;
    }
    return false;
}
REPAIR 

bool fooOK( char ch )
{
    if( (unsigned char) ch == 0xFF ) {  // OK
        return true;
    }
    return false;
}
