User-specified identifiers (internal and external) will not rely on significance of more than 64 characters [PORT-25-2]

DESCRIPTION 

Identifiers names consisting of more than 64 characters should not be used.
SINCE 

v7.1
BENEFITS 

Rule ensures that code can be ported between the majority of compilers/linkers
without requiring modification (shortening) of identifiers names
EXAMPLE 

void foo(int really_long_paaaaaaaaaaaaaaaaraaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaam) // Violation 65 chars
{

}
REPAIR 

void foo(int really_long_paaaaaaaaaaaaaaaaraaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaam)  // OK 64 chars
{

}
REFERENCES 

JOINT STRIKE FIGHTER, AIR VEHICLE, C++ CODING STANDARDS
Chapter 4.9 Style, AV Rule 46
