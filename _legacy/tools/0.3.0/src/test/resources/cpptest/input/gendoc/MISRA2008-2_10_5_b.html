The identifier name of a non-member object or function with static storage duration should not be reused [MISRA2008-2_10_5_b-4]

DESCRIPTION 

Object or function identifier with static storage duration should be unique.
BENEFITS 

Reuse of identifier names can lead to errors and confusion.
EXAMPLE 

static float a;
static void foo()
{
static int a ;  /* Violation */
static int foo; /* Violation */  
}
REPAIR 

Do not use the same names for different static object or functions.
REFERENCES 

1. MISRA-C:2004 Guidelines for the use of the C language in critical systems
   Chapter 6, Section 5

2. MISRA C++:2008 Guidelines for the use of the C++ language in critical
   systems, Chapter 6, Section 2, Rule 2-10-5
